# Shadows of the Knight 1

https://www.codingame.com/training/medium/shadows-of-the-knight-episode-1

## Objectif

Trouver une position inconnue dans une matrice 2D, à l'aide d'indices directionnels (U, UR, R, DR, D, DL, L ou UL) pour réduire progressivement la zone de recherche.

### Entrée de départ

- w, h la taille de la matrice
- x0, y0 la position de départ
- n le nombre max de tours autorisé

## Notions

- Recherche dichotomique
- Intervalles
